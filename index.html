Embedded Device for Audio Signal Analysis and Classification of Musical Notes Using Machine Learning

เนื่องจากในการฝึกซ้อมเล่นดนตรี ผู้เล่นไม่สามารถรับรู้ได้ว่าเล่นได้ถูกต้องสมบูรณ์ตามโน๊ตเพลงหรือไม่ จึงต้องการตัวช่วยในการบอกว่า เล่นได้ถูกต้องขนาดไหน รวมถึงการฟังเพลงที่ชื่อชอบ และต้องการเล่นเพลงนั้น แต่ไม่สามารถหาโน้ตเพลงเพื่อมาเล่นได้ จึงต้องการตัวช่วยในการเขียนโน้ตจากเสียงที่ได้ฟัง
